# 🏃 Runner Tracker

A simple, clean, and motivating running tracker that helps you log runs, track progress, and stay consistent.

---

## ✨ Features

- 📊 **Runs Dashboard**
  - Monthly stats (distance, runs, streak)
  - View all runs with pace, time, and distance at a glance

- ➕ **Log a Run**
  - Quick and focused input
  - Track:
    - Distance
    - Time
    - Effort (easy / good / tough)
    - Optional notes
  - Automatic pace calculation

- 📈 **Progress Tracking**
  - Monthly comparison (vs last month)
  - 6-month distance chart
  - All-time stats:
    - Longest run
    - Best pace
    - Total distance & time

- 🧹 **Clean UX**
  - Long-press to delete runs
  - Minimal, distraction-free interface
  - Warm and encouraging design

- 💾 **Local Storage**
  - All data saved on your device
  - No login required
